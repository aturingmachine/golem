# report 

Report a bug/issue or provide a suggestion to the admin of this Golem Bot.

## Examples

### Legacy Command

```
$go report 
```

### Slash Command

```
/goreport
```

## Arguments
- **reportmessage*** - `string`: A description of the bug/issue/suggestion.
