# skip <badge text="LocalMusic*" type="localmusic-badge optional-mod-badge tooltip-root"/> <badge text="Youtube*" type="youtube-badge optional-mod-badge tooltip-root"/>

Skip queued tracks.

## Examples

### Legacy Command

```
$go skip
$skip
```

### Slash Command

```
/goskip
```

## Arguments
- **skip-count** - `integer`: The number of tracks to skip.

```
