# search <badge text="Music" type="music-badge" />

Search for a local track and view the result set.

## Examples

### Legacy Command

```
$go search twice tt
```

### Slash Command

```
/gosearch twice tt
```

## Arguments
- **query*** - `string`: The query to run against the Local Search Index.
